# A2UI Salesforce Agent Sample

This sample uses the Agent Development Kit (ADK) to create a "Salesforce" agent.

## Prerequisites

- Python 3.13 or higher
- [UV](https://docs.astral.sh/uv/)
- GEMINI_API_KEY environment variable

## Running the Sample

1. Navigate to the directory:

    ```bash
    cd samples/agent/adk/adk_salesforce
    ```

2. Run the server:

   ```bash
   uv run .
   ```

The server listens on port 10003 by default.
